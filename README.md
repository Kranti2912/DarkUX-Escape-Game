# DarkUX-Escape-Game
Dark Pattern Escape is a fast, playful browser game where you’re trapped inside deceptive interfaces and must find the real way out. Each level uses sneaky UX tricks—misleading buttons, hidden exits, fake confirmations—to confuse you. Click carefully, avoid traps, and escape the manipulation before it gets worse.
Game link: https://kranti2912.github.io/DarkUX-Escape-Game/
